# Security Policy

## Supported Versions

Currently, this project is considered feature-complete. While no new major versions or feature updates are planned, we remain committed to maintaining the security of the existing release.

| Version | Supported          | Notes                                     |
| ------- | ------------------ | ----------------------------------------- |
| 1.0.0   | :white_check_mark: | Active security monitoring and patching.  |
| < 1.0.0 | :x:                | No longer supported. Please update.       |

## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have discovered a security vulnerability, please report it to us privately to protect the user community.

### How to Report

Please do **not** use the public issue tracker for security-related reports. Instead, send a detailed email to:

**mihail.a.minchev2030@pmg3-varna.org**

To help us evaluate and fix the issue as quickly as possible, please include:

* **Type of issue:** (e.g., Buffer overflow, XSS, SQL injection).
* **Location:** File names, lines of code, or specific functionality affected.
* **Steps to Reproduce:** Clear instructions or a Proof of Concept (PoC).
* **Impact:** What an attacker could achieve by exploiting this bug.

### Our Commitment

If you follow the reporting process above, we commit to the following:

* **Acknowledgment:** We will acknowledge receipt of your report within **3–5 business days**.
* **Confidentiality:** We will keep information about the vulnerability confidential until a fix is released.
* **Resolution:** We will work to investigate and resolve the issue in a timely manner.
* **Credit:** We will provide credit to the researcher in our release notes, unless you prefer to remain anonymous.

## Responsible Disclosure

We ask that you practice responsible disclosure by giving us a reasonable amount of time to fix the issue before making any information public. This helps ensure the safety of everyone using this project.

Thank you for helping keep this project secure!
